HDRL-LF is a hybrid intelligent VM placement framework designed for energy-efficient cloud data center management. It integrates three powerful components—LSTM-based workload forecasting, KMeans-driven VM sensitivity classification, and a Dueling Deep Q-Network (DQN) placement agent—to deliver adaptive and proactive resource allocation. The framework automatically loads real-world datasets (Bitbrains, Google Cluster), preprocesses CPU utilization traces, trains the forecasting model, evaluates performance using multiple metrics (MSE, MAE, RMSE, MAPE, R²), and visualizes results. HDRL-LF serves as a research-ready and extensible architecture for intelligent cloud orchestration, enabling reproducible experiments, comparative benchmarking against heuristics, and end-to-end ML-driven decision-making for dynamic VM placement.
